# Date
To use [Date](https://github.com/HowardHinnant/date) in [ZPM](http://zpm.zefiros.eu), just use:

# [ZPM](http://zpm.zefiros.eu) Installation
In `.package.json`
```json
"requires": [
		{
			"name": "Zefiros-Software/Date",
			"version": ">=2.1.0"
		}
]
```

In `zpm.lua`
```lua
zpm.uses "Zefiros-Software/Date"
```
